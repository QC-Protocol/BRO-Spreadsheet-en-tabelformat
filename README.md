# BRO-Spreadsheet-en-tabelformat
Format voor uitwisseling van bron- en BRO-data.
Het format voldoet aan de kwaliteitseisen (IMBRO en IMBRO\A) van de Basisregistratie Ondergrond (BRO) en dekt de inhoud ervan af.
De tabellen of spreadsheets zijn gebaseerd op een vereenvoudiging van de onderliggende datamodellen en gegevenscatalogi.
