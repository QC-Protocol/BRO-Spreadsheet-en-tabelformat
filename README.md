# BRO-Spreadsheet-en-tabelformat
Format voor uitwisseling van bron- en BRO-data.
Het format voldoet aan de eisen van de Basisregistratie Ondergrond (BRO) en dekt de inhoud ervan af (IMBRO en IMBRO\A) 
De tabellen of spreadsheets zijn gebaseerd op een vereenvoudigde maar complete versie van de onderliggende datamodellen en gegevenscatalogi
